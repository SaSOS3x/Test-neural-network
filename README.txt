Крч, это тестовая нейросеть для:

1. Проверки разных функций активаций нейрона;
2. Архитектур нейронок;
3. Теста и разработки новых методов обучения и т.д.

Проект:

Файл с бд - db.db, это база данных sqlite3

learn.py - файл для обучения и переобучения сети, полученный синаптический вес
сохраняется в бд в виде бинарных данных

setup.py - файл для запуска нейросети для решения поставленной задачи
синаптический вес берется из бд, бмнарные данные преобразуются в массив numpy
после чего и выполняются вычисления.



Вообщем, весь этот интелект, ничто иное, как переход функции к определенным числам,
на основе которых выполняются вычисления с результатом: число от 0 до 1 с плавающей точкой

В свою очередь, эти самые числа подбираются автоматически в зависимости алгоритма
и вычислений заложенных в программу
